## 餐廳列表網站
該網站利用 node.js、express 以及 handlebars 完成

## 環境建置需求
* 執行環境： node.js v10.15.0
* 框架：express 
* 模板引擎： express-handlebars

## 安裝與下載
#### 專案下載
    git clone https://github.com/a227799770055/restaurant_list.git
#### 安裝 package
    npm install
#### 使用 nodemon 啟動伺服器
    npm run dev

## 網站功能
* 點擊餐廳清單可以查看餐廳細節（評分、簡介、地址、電話）
* 利用關鍵字（名稱或是種類）可以搜尋餐廳
